dotfiles
========
## requirements

- vim
- zsh
- git
- gcc
- make
- cmake
- tmux >= 0.7
- rbenv

## setup

```
$ git clone git://github.com/YuuShigetani/dotfiles.git ~/dotfiles
$ chmod +x ~/dotfiles/setup.sh
$ ~/dotfiles/setup.sh
```

## uninstall

```
$ chmod +x ~/dotfiles/uninstall.sh
$ . ~/dotfiles/uninstall.sh
```
