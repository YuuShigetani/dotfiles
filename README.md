dotfiles
========
## Requirements

- zsh
- tmux >= 1.7
- git
- mercurial
- ruby
- fontforge

### Optional

- boxen (for mac)
- cmake (tmux-mem-cpu-load)
- lua (neocomplete)

## Setup

```
$ git clone git://github.com/calorie/dotfiles.git ~/dotfiles
$ cd ~/dotfiles
$ ./script/setup
```

### Installed packages

- ricty (font)
- vvm-rb (vim version manager)
- auto-fu
- powerline-shell
- tmux-powerline
- tmux-mem-cpu-load
- tmuxinator
- some vim plugins

## Uninstall

```
$ cd ~/dotfiles
$ ./script/uninstall
```
