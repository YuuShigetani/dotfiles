dotfiles
========
## requirements

- zsh
- tmux >= 1.7
- git
- mercurial
- ruby
- fontforge

### optional

- boxen (for mac)
- cmake (tmux-mem-cpu-load)
- lua (neocomplete)

## setup

```
$ git clone git://github.com/calorie/dotfiles.git ~/dotfiles
$ cd ~/dotfiles
$ ./setup.sh
```

### installed pkg

- ricty
- vvm(vim version manager)
- auto-fu
- powerline-shell
- tmux-powerline
- tmux-mem-cpu-load
- tmuxinator
- some vim plugins

## uninstall

```
$ cd ~/dotfiles
$ ./uninstall.sh
```
