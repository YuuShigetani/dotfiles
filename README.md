dotfiles
========
## requirements

- vim
- zsh
- git
- gcc
- make
- cmake
- tmux >= 0.7
- rbenv
- ruby

## setup

```
$ git clone git://github.com/YuuShigetani/dotfiles.git
$ cd dotfiles
$ chmod +x setup.sh
$ ./setup.sh
```

## uninstall

```
$ cd /path/to/dotfiles
$ chmod +x uninstall.sh
$ ./uninstall.sh
```
