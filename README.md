dotfiles
========
## requirements

- zsh
- tmux >= 1.7
- git
- mercurial
- python
- fontforge

### optional

- brew (for mac)
- cmake (tmux-mem-cpu-load)
- lua (neocomplete)
- npm (jsctags)

## setup

```
$ git clone git://github.com/calorie/dotfiles.git ~/dotfiles
$ cd ~/dotfiles
$ ./setup.sh
```

### installed pkg

- ricty
- vvm(vim version manager)
- rbenv
- auto-fu
- tmux-powerline
- tmux-mem-cpu-load
- tmuxinator
- some vim plugins

## uninstall

```
$ cd ~/dotfiles
$ ./uninstall.sh
```
